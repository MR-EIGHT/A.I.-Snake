# A.I. Snake

This project is a classic-snake-game implemented in Python with heuristic algorithms.

### Usage:

Install required libraries by running the command below:

```
pip install -r requirements.txt
```

Then, run the UI:

```
python UI.py
```

### Requirements:

* pygame
* pygame-menu
* numpy
